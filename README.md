Library Management (Python)

A simple command line library management system built in Python to practice
object oriented programming (OOP), data structures, and user interaction.

Features:
- Add, show, find, and remove books
- Work with classes (`Book`, `Library`)
- Case-insensitive search
- Interactive CLI menu

 Skills shown:
- Classes, methods, and lists
- Loops and conditional statements
- Input/output handling in Python

Run locally

```bash
python library.py
```

Example

```
1. Add Book
2. Show Books
3. Find Book
4. Remove Book
5. Exit
```
